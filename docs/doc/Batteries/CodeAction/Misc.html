        In the following:

        instance : Monad Id := _

        instance : Monad Id where
  map := _
  mapConst := _
  pure := _
  seq := _
  seqLeft := _

              def Batteries.CodeAction.removeAfterDoneAction :
              TacticCodeAction

              The "Remove tactics after 'no goals'" code action deletes any tactics following a completed proof.

              example : True := by
  trivial
  trivial -- <- remove this, proof is already done
  rfl

              is transformed to

              example : True := by
  trivial

                    def Batteries.CodeAction.casesExpand :
                    TacticCodeAction

                    Invoking tactic code action "Generate an explicit pattern match for 'induction'" in the following:

                    example (x : Nat) : x = x := by
  induction x

                    produces:

                    example (x : Nat) : x = x := by
  induction x with
  | zero => sorry
  | succ n ih => sorry

                    It also works for cases.
