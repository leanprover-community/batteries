              def Lean.Meta.setMVarUserNamesAt (e : Expr) (isTarget : Array Expr) :
              MetaM (Array MVarId)

              Auxiliary method for (temporarily) setting the user facing name of metavariables. Let ?m be a metavariable in isTarget.contains ?m, and ?m does not have a user facing name. Then, we try to find an application f ... ?m in e, and (temporarily) use the corresponding parameter name (with a fresh macro scope) as the user facing name for ?m. This method returns all metavariables whose user facing name has been updated.

                  MetaM Expr

                  Similar to mkForallFVars, but tries to infer better binder names when xs contains metavariables. Let ?m be a metavariable in xs s.t. ?m does not have a user facing name. Then, we try to find an application f ... ?m in the other binder type and type, and (temporarily) use the corresponding parameter name (with a fresh macro scope) as the user facing name for ?m.
