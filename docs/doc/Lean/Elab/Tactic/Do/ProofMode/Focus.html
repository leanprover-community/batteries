    structure Lean.Elab.Tactic.Do.ProofMode.FocusResult :
    Type

    The result of focusing the context of a goal goal : MGoal on a particular hypothesis. The focused hypothesis is returned as focusHyp : Expr, along with the residual restHyps : Expr and a proof : Expr for the property goal.hyps ⊣⊢ₛ restHyps ∧ focusHyp.

    • focusHyp : Expr
    • restHyps : Expr
    • proof : Expr
