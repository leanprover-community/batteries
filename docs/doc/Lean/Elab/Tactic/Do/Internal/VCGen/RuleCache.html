        def Lean.Elab.Tactic.Do.Internal.VCGen.mkLatticeOpRuleCached (rhs : Expr) (op : LatticeOp) :
        VCGenM Meta.Sym.BackwardRule

        Cached construction of a lattice-split backward rule for the operator heading rhs. On a cache miss the rewrite and terminal sets are assembled from the built-in seeds and the operator's @[frameproc] contributions (fp?); a cache hit skips that work.

        Cache key: the head … cₙ prefix built from the constant arguments (which the rule bakes in verbatim) and the argument count (which fixes the schematic operand and state count).
