    structure Lean.ExtraModUse :
    Type

    Additional import dependency for elaboration.

    • module : Name

      Dependency's module name.

    • isExported : Bool

      Whether dependency must be imported as public.

    • isMeta : Bool

      Whether dependency must be imported as meta.
