    (α × β → φ) → α → β → φ

    Transforms a function from pairs into an equivalent two-parameter function.

    Examples:

    • Function.curry (fun (x, y) => x + y) 3 5 = 8
    • Function.curry Prod.swap 3 "five" = ("five", 3)
    Equations
    • Function.curry f a b = f (a, b)
