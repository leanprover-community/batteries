    Polymorphic slices #

    This module provides slices -- views on a subset of all elements of an array or other collection, demarcated by a range of indices.

    • Init.Data.Slice.Basic defines the Slice structure. All slices are of this type.
    • Init.Data.Slice.Operations provides functions on Slice via dot notation. Many of them are implemented using iterators under the hood.
    • Init.Data.Slice.Notation provides slice notation based on ranges, relying on the Sliceable typeclass.
    • Init.Data.Slice.Array provides the Sliceable instance for array slices.
