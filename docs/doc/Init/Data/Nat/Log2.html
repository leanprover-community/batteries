    Base-two logarithm of natural numbers. Returns ⌊max 0 (log₂ n)⌋.

    This function is overridden at runtime with an efficient implementation. This definition is the logical model.

    • Nat.log2 1 = 0
    • Nat.log2 2 = 1
    • Nat.log2 4 = 2
    • Nat.log2 7 = 2
