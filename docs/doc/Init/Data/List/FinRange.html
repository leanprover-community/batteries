    def List.finRange (n : Nat) :
    List (Fin n)

    Lists all elements of Fin n in order, starting at 0.

    Examples:

    • List.finRange 0 = ([] : List (Fin 0))
    • List.finRange 2 = ([0, 1] : List (Fin 2))
    Equations
    • List.finRange n = List.ofFn fun (i : Fin n) => i
