    @[inline]
    def Fin.addNat? {n : Nat} (i : Fin n) (m : Nat) :
    Option (Fin n)

    Overflow-aware addition of a natural number to an element of Fin n.

    Examples:

    • (2 : Fin 3).addNat? 1 = (none : Option (Fin 3))

    • (2 : Fin 4).addNat? 1 = (some 3 : Option (Fin 4))
