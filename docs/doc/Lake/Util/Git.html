        def Lake.Git.filterUrl? (url : String) :
        Option String

        Try to turn a remote URL into a URL that can be used to, e.g., make GitHub API requests. That is, do not accept SSH URLs and drop an ending .git.

        Equations
        • Lake.Git.filterUrl? url = if url.startsWith "git" = true then none else if url.endsWith ".git" = true then some (url.dropEnd 4).copy else some url
