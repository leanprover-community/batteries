    Red-Black Dictionary #

    Defines an insertion-ordered key-value mapping backed by an red-black tree. Implemented via a key-index Std.TreeMap into an Array of key-value pairs.

    source
    structure Lake.Toml.RBDict (α : Type u) (β : Type v) (cmp : α → α → Ordering) :
    Type (max u v)
    • items : Array (α × β)
    • indices : Std.TreeMap α Nat cmp
