    Body.Basic #

    This module defines the Body typeclass for HTTP body streams, and shared conversion types ToByteArray and FromByteArray used for encoding and decoding body content.

    source
    class Std.Http.Body (α : Type) :
    Type

    Typeclass for values that can be read as HTTP body streams.

    • recv : α → Async.Async (Option Chunk)

      Receives the next body chunk. Returns none at end-of-stream.

    • close : α → Async.Async Unit

      Closes the body stream.

    • isClosed : α → Async.Async Bool

      Returns true when the body stream is closed.

    • recvSelector : α → Async.Selector (Option Chunk)

      Selector that resolves when a chunk is available or EOF is reached.

    • tryRecv (body : α) : Async.Async (Option (Option Chunk))

      Non-blocking receive attempt. Returns none if no chunk is immediately available, some (some chunk) when a chunk is ready, or some none at end-of-stream.

    • getKnownSize : α → Async.Async (Option Length)

      Gets the declared size of the body.

    • setKnownSize : α → Option Length → Async.Async Unit

      Sets the declared size of a body.
