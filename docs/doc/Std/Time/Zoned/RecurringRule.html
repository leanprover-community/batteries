    source
    inductive Std.Time.TimeZone.TransitionSpec :
    Type

    Specifies how a DST transition date is expressed.

    • mwd (month : Month.Ordinal) (week : Week.Aligned.Ordinal) (day : Weekday.Ordinal) : TransitionSpec

      Mm.w.d form: month m, week w (1–5; 5 = last), ISO weekday d.

    • julian (day : Internal.Bounded.LE 1 365) : TransitionSpec

      Jn form: Julian day 1–365 (Feb 29 never counted, even in leap years).

    • julian0 (day : Internal.Bounded.LE 0 365) : TransitionSpec

      n form: zero-based day 0–365 (Feb 29 counted in leap years).
