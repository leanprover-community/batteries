    structure Std.Time.PlainDateTime :
    Type

    Represents a date and time with components for Year, Month, Day, Hour, Minute, Second, and Nanosecond.

    • date : PlainDate

      The Date component of a PlainDate

    • time : PlainTime

      The Time component of a PlainTime
